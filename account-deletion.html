계정 삭제 안내

시행일: 2026년 5월 18일  |  최종 수정: 2026년 5월 18일

AnglerDex(이하 "서비스")는 이용자의 데이터 자기결정권을 보장합니다. 본 페이지는 계정 및 관련 데이터의 삭제(탈퇴) 요청 방법과 삭제·보관되는 데이터의 범위를 안내합니다.
1 서비스 및 운영자 정보

2 계정 삭제 요청 방법

이용자는 다음 두 가지 방법 중 하나로 계정 삭제(탈퇴)를 요청할 수 있습니다.

방법 A (권장)

앱 내에서 탈퇴

로그인 가능한 경우 가장 빠르고 안전한 방법입니다. 즉시 처리됩니다.

방법 B

이메일로 요청

앱 접근이 어려운 경우 운영자에게 직접 이메일을 보내주세요. 영업일 기준 3일 이내 처리됩니다.

방법 A — 앱 내 탈퇴 단계

  1. 앱 실행 후 로그인 AnglerDex 앱을 열고 본인 계정으로 로그인합니다 (Google / 카카오 / Apple).
  2. "MY" 탭 진입 하단 탭바 우측의 "MY" 탭을 누릅니다.
  3. 우측 상단 메뉴 → "마이페이지" 선택 햄버거 메뉴(≡) 아이콘을 누른 뒤 "마이페이지" 항목을 선택합니다.
  4. "회원 탈퇴" 버튼 탭 마이페이지 하단의 "회원 탈퇴" 항목을 누릅니다.
  5. 안내 확인 후 최종 동의 삭제될 데이터 항목 안내를 확인하고 "예, 탈퇴하겠습니다"를 누르면 즉시 계정과 관련 데이터가 삭제됩니다.

방법 B — 이메일 요청 단계

  1. 이메일 작성 본인이 가입에 사용한 이메일 주소(또는 카카오/Apple 가입 시 사용한 식별 정보)로 이메일을 작성합니다.
  2. 제목: "AnglerDex 계정 삭제 요청" 본문에 가입 시 사용한 소셜 로그인 종류(Google / 카카오 / Apple)와 닉네임을 기재해 주세요. 본인 확인을 위해 필요합니다.
  3. 수신처: jjoraey@gmail.com 위 이메일로 전송하시면 영업일 기준 3일 이내에 본인 확인 절차 후 계정 및 관련 데이터를 삭제하고 처리 결과를 회신해드립니다.
본인 확인은 가입 시 사용한 이메일 주소와의 일치 여부, 또는 앱 내 닉네임·고유 식별자를 통해 진행됩니다. 본인 확인이 불가능한 경우 삭제 요청이 거부될 수 있습니다.
3 삭제되는 데이터 및 보관 데이터

계정 삭제 시 다음과 같이 데이터가 처리됩니다.

데이터 항목 처리 방식 보관 기간
계정 정보 (이메일, 닉네임, 프로필 사진, UID) 즉시 삭제 없음
낚시 기록 (어종, 길이, 위치, 사진, 환경 데이터) 즉시 삭제 없음
도감 진행도 (해금한 어종 목록) 즉시 삭제 없음
피드 게시물, 댓글, 좋아요 기록 즉시 삭제 없음
나의 장비함 (낚싯대, 릴, 채비 등) 즉시 삭제 없음
업로드 사진 파일 (Firebase Storage) 즉시 삭제 없음
푸시 알림 토큰 (FCM) 즉시 삭제 없음
서비스 이용 로그 (접속 기록 등) 법령상 보관 3개월 (통신비밀보호법)
신고·민원 처리 기록 (해당 시) 법령상 보관 3년 (전자상거래법)

"즉시 삭제" 항목은 탈퇴 요청 처리와 동시에 Firebase Authentication, Firestore, Storage에서 영구적으로 파기되며 복구할 수 없습니다.

"법령상 보관" 항목은 관계 법령(통신비밀보호법, 전자상거래법 등)이 정한 최소 기간 동안만 보관되며, 보관 기간 종료 즉시 복구 불가능한 방법으로 파기됩니다. 보관 기간 동안에도 마케팅, 통계, 제3자 제공 등 다른 목적으로는 이용되지 않습니다.

4 계정 유지하며 일부 데이터만 삭제하려면

계정은 유지한 채 특정 데이터만 삭제하고 싶은 경우 앱 내에서 다음과 같이 처리할 수 있습니다.

  • 개별 낚시 기록 삭제: 도감 → 해당 기록 진입 → 우측 상단 메뉴 → "삭제"
  • 개별 피드 게시물 삭제: 피드 → 본인 게시물 → 우측 상단 메뉴 → "삭제"
  • 개별 댓글 삭제: 댓글 좌측 슬라이드 → "삭제"
  • 장비 삭제: MY → 나의 장비함 → 해당 장비 → 삭제
  • 프로필 사진 변경/삭제: MY → 프로필 사진 탭 → 변경 또는 기본 이미지로 재설정

위 방법으로 처리되지 않는 데이터의 부분 삭제는 운영자 이메일(jjoraey@gmail.com)로 별도 요청하시면 검토 후 처리해 드립니다.

5 탈퇴 전 확인 사항

계정 삭제는 다음과 같은 결과를 동반하므로 신중히 결정해 주세요.

  • 그동안 등록한 낚시 기록, 도감 진행도, 어종별 최대어 기록이 모두 사라지며 복구할 수 없습니다.
  • 명예의 전당에 등재된 본인의 기록도 함께 삭제되어 랭킹에서 제외됩니다.
  • 피드에 작성한 게시물·댓글이 모두 삭제되며, 다른 이용자가 본인 게시물에 남긴 댓글도 함께 사라집니다.
  • 동일한 소셜 계정으로 재가입은 가능하지만, 이전 데이터는 복원되지 않습니다.
법령상 보관 항목(서비스 이용 로그, 신고 처리 기록)은 탈퇴 후에도 위 표에 명시된 기간 동안 별도 보관되며, 보관 기간이 끝나는 즉시 자동으로 파기됩니다. 이 기간 중에도 본인 또는 수사기관의 정당한 요청 외에는 어떤 목적으로도 이용되지 않습니다.
